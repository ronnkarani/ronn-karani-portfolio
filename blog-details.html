🔧 How I Built My Portfolio

July 2025 Ronny 321 views 45 likes 3 comments

I built this personal portfolio to showcase my work, skills, and journey as a full-stack developer. I started with basic HTML and CSS, then gradually introduced JavaScript animations and particles.js for flair.

The site includes sections for projects, blog posts, and contact, with a dark-themed UI and mobile responsiveness. I also used a modular approach to keep my codebase clean.

Key Features:

  • Particles.js animated background
  • Dark/light mode (coming soon)
  • Responsive layout using Flex/Grid
